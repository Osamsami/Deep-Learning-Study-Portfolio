# Multilayer Perceptron (MLP) & Backpropagation

## Overview
In This Study we  implements a Multilayer Perceptron (MLP) from scratch using NumPy to solve the XOR problem. It demonstrates forward propagation, backpropagation, and non-linear learning.

## Objectives
- Understand MLP architecture
- Implement forward propagation
- Implement backpropagation
- Solve XOR problem
- Visualize learning process

## Architecture
Input Layer: 2 neurons  
Hidden Layer: 4 neurons  
Output Layer: 1 neuron  

Structure: 2 → 4 → 1

## Key Concepts
- Non-linearity using Sigmoid
- Gradient Descent
- Error Backpropagation

## Results
The model successfully learns XOR, which a single-layer perceptron cannot solve.
