# Visualizations

This folder contains graphical outputs generated during model training and evaluation.

## Files

### training_loss.png
Shows how loss decreases over epochs.

### training_accuracy.png
Shows how model accuracy improves during training.

### correct_vs_incorrect.png
Displays number of correct vs incorrect predictions.

### class_accuracy.png
Shows performance for each room category.

### prediction_grid.png
Displays actual vs predicted labels for sample images.

### single_prediction.png
Shows prediction for a single image.

### sample_images.png
Displays dataset sample images.

---

## Purpose
These visualizations help in understanding model performance and identifying limitations of MLP.
